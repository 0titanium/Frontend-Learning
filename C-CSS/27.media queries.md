Q.<br/>
미디어 쿼리의 기본구문은?
<br/><br/>

A.<br/>
```css
@media media-type and (media-feature-rule) {
    /* CSS rules */
}
```
<br/><br/>

Q.<br/>
지정할 수 있는 미디어 유형은?
<br/><br/>

A.<br/>
1. all
2. print

4. speach

Q.<br/>
복잡한 미디어 쿼리를 작성하는 방법은?
<br/><br/>

A.<br/>
1. and (&&)로 미디어 기능을 결합한다.
2. , (||)로 미디어 기능을 분리한다.
3. not (!)으로 미디어 쿼리를 부정한다.
<br/><br/>
